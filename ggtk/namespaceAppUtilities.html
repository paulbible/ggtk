Detailed Description

The AppUtilities namespace provides utility functions that facilitate application creation and integration.

This namespace defines free functions that aid in constructing applicaitons using this tool kit. The main use for this name space is to construct and check the validity of paramaters used as intput to GO applicaitons and to provide certain container converstion (vector -> set etc.).
