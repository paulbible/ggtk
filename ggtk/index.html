Introduction

GGTK is a set of modular tools for working with Gene Ontology Graphs. GGTK is composed of a set of C++ header libraries. Simply include the necessary files in your C++ source code. Some header only GGTK is composed of a set of C++ header libraries. Simply include the necessary files in your C++ source code. Some header only Boost libraries are required for GGTK (Boost 1.54 or later). Using header only libraries allows scientist and developers to use as much or as little of GGTK as needed.

Getting Started

Since GGTK soruce code is header only simply #include the needed files into your project. These documentation papes provide the API for the GGTK's C++ code. More information can by found in the Python Documentation along with some examples uses.
