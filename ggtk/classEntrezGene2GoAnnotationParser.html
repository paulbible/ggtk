Detailed Description

A class to parse an Entrez gene2go annotation file.

This class will read an Entrez gene2go file and add those annoations to an AnnotationData class. Available at: ftp://ftp.ncbi.nih.gov/gene/DATA/

Implements AnnotationParserInterface
