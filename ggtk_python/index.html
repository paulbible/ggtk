GGTK: The GO Graph Tool Kit¶

GGTK is a set of modular tools for working with the Gene Ontology Graph. These pages provide documentation for the Python interface for GGTK. The C++ documentation provides details about the GGTK API. The GitHub Project hosts all the code for GGTK.

Getting Started¶

The major requirements for using the Python interface to GGTK are Python 2.7, the setuptools package, a c++ compiler, and the Boost C++ Libraries version 1.54 or higher. Windows user may need to download a C++ compiler that is compatible with their Python release. See here for more information.

GGTK requires that you set the environment variable BOOST to the location of the Boost directory.:

export BOOST=/path/to/boost_1_54_0

At this point, you may want to unzip the example graph and annotation data.:

cd example_graphs
gunzip *.gz
cd ../example_annotations
gunzip *.gz
cd ..

Once you have downloaded the GGTK project folder. Enter the python diretory and enter the following commands:

cd python
python setup.py build
python setup.py install

Using the --user option lets you install the package in your home directory.:

python setup.py install --user

After Python has built and installed the package you may optionally run the tests. You may need to unzip the examples files.:

cd tests
python test_all.py

These tests will take about 5-10 minutes to complete.

After than you are all set! Take a look at the example code in the next section.
