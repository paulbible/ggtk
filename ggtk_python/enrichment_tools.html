Enrichment Tools¶

These functions are useful in determining if a GO term is enriched within a set of genes given a population.

EnrichmentTools Module¶

EnrichmentTools Module This module defines free functions that allow enrichment p-values to be calculated. These funcitons can serve as the foundation for more sophisticated enrichment anlayis.

ggtk.EnrichmentTools.enrichmentSignificance(go_graph, anno_data, genes, go_term)[source]¶

This method performs a hypergeometic test of enrichment for a term given a set of genes that serves as the sample. The population is taken as all genes in the annotation database.

ggtk.EnrichmentTools.getDescendantGenes(go_graph, anno_data, go_term)[source]¶

This method calculates the set of the genes annotated with a given term or transatively with a child of that term.

ggtk.EnrichmentTools.oneSidedRawPvalue_hyper(sample, pop_success, population, test_value)[source]¶

This method calculates p-value of a hypergeometice test give 4 values.

The sample size, n The population success K The the population size N The test value k

Answers the question: “What is probability of seeing value of k or more successes in a sample of size n, given that the population of size N contains K total successes.”
