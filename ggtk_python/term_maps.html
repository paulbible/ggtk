Term Maps¶

Term Maps pefroms some mapping between a GO term and some other information. GGTK defines term maps for probability, information content, and ontology depth.

BaseTermMap Class¶

class ggtk._BaseTermMap.BaseTermMap(map_proxy)[source]¶

Bases: object

An abstract map class for go term maps Since the probability, information content, and depth maps will have the same inteface, this class should reduce redundant code.
